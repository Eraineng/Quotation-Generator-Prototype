🧾 Quotation Generator Prototype

A web-based application for generating quotations quickly and efficiently. Built with modern frontend technologies to streamline pricing workflows and improve productivity.

🚀 Features
📄 Create and manage quotations
✏️ Editable line items (products/services)
💰 Automatic price calculations
📦 Clean and modular UI components
⚡ Fast and responsive interface
🧩 Built with scalable architecture
🛠️ Tech Stack
Framework: Next.js
Language: TypeScript
Styling: Tailwind CSS
UI Components: Custom / ShadCN-style components
Tooling: ESLint, PostCSS
📂 Project Structure
.
├── app/                # App routes and pages
├── components/         # Reusable UI components
├── lib/                # Utility functions
├── public/             # Static assets
├── package.json        # Dependencies and scripts
└── tsconfig.json       # TypeScript config
⚙️ Installation

Clone the repository:

git clone https://github.com/Eraineng/Quotation-Generator-Prototype.git
cd Quotation-Generator-Prototype

Install dependencies:

npm install
▶️ Running the App

Start development server:

npm run dev

Then open:

http://localhost:3000
🧪 Scripts
npm run dev      # Start development server
npm run build    # Build for production
npm run start    # Run production build
npm run lint     # Lint code
